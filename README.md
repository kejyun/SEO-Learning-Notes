## 說明

### 搜尋引擎最佳化（Search Engine Optimization，簡稱SEO）

又稱為搜尋引擎優化，是一種透過了解搜尋引擎的運作規則來調整網站，以期提高目的網站在有關搜尋引擎內排名的方式。
